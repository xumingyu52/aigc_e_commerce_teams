# 启动命令总表

这份文档只做一件事：把项目里常用的启动命令集中到一起，让第一次接手项目的人直接照着跑。

## 一、先理解这几个端口

- `5000`：主业务后端 API
- `6000`：页面壳和兼容代理层
- `8000`：Qwen3-TTS 服务
- `8001`：Qwen3-ASR 服务
- `10003`：前端面板 WebSocket
- `10004`：数字人运行时 WebSocket

## 二、统一原则

- 所有命令都建议在项目根目录执行
- 所有终端都先激活虚拟环境
- 推荐开 3 个终端窗口分别启动 TTS、ASR、主程序

项目根目录：

```powershell
cd E:\aigc_team\aigc_e_commerce_teams
```

激活虚拟环境：

```powershell
.\.venv\Scripts\Activate.ps1
```

## 三、标准启动顺序

### 1. 启动 TTS 服务

终端 1：

```powershell
cd E:\aigc_team\aigc_e_commerce_teams
.\.venv\Scripts\Activate.ps1
python .\tts\qwen3tts_server\server.py
```

启动成功后，TTS 服务默认监听：

```text
http://127.0.0.1:8000
```

可选健康检查：

```powershell
curl http://127.0.0.1:8000/health
```

### 2. 启动 ASR 服务

终端 2：

```powershell
cd E:\aigc_team\aigc_e_commerce_teams
.\.venv\Scripts\Activate.ps1
$env:PYTHONPATH="."
python .\asr\qwen3_server\server.py
```

启动成功后，ASR 服务默认监听：

```text
http://127.0.0.1:8001
```

可选健康检查：

```powershell
curl http://127.0.0.1:8001/health
```

### 3. 启动主程序

终端 3：

```powershell
cd E:\aigc_team\aigc_e_commerce_teams
.\.venv\Scripts\Activate.ps1
python .\main.py
```

主程序启动后会拉起：

- 登录服务
- Dashboard 服务
- 主 API 服务 `5000`
- 页面壳/兼容层 `6000`
- WebSocket 服务 `10003` / `10004`

## 五、访问入口

- 主 API：`http://127.0.0.1:5000`
- 页面壳：`http://127.0.0.1:6000`
- Live2D 前端开发页通常由 `main.py` 启动的 frontend 提供

如果你要调试直播主链，优先看 `5000`。

## 六、配置文件要看哪几个

### 1. `system.conf`

至少确认这些配置：

```ini
[key]
ASR_mode = qwen3
tts_module = qwen3
qwen3_tts_url = http://127.0.0.1:8000/tts
qwen3_asr_url = http://127.0.0.1:8001/asr
backend_api_url = 127.0.0.1:5000
start_mode = web
```

### 2. `config.json`

重点看：

- `attribute.voice`

当前建议音色：

- `Qwen3-灵动女声`
- `Qwen3-稳重男声`

## 八、常见错误

### 1. `.venv\Scripts\python.exe` 找不到

原因通常是你已经切进了子目录，还在用根目录相对路径。

最稳妥方式：

- 先回项目根目录
- 激活虚拟环境后直接用 `python`

### 2. 页面打开了，但没有声音

优先检查：

- `8000` 是否启动
- `system.conf` 中 `tts_module=qwen3`
- `config.json` 中音色是否有效

### 3. 录音后没识别结果

优先检查：

- `8001` 是否启动
- `ASR_mode=qwen3`
- `qwen3_asr_url` 是否正确

### 4. 前端能访问，接口报 502 或连接失败

优先检查：

- `5000` 主 API 是否已经由 `main.py` 启动
- `6000` 只是兼容代理层，主业务还是看 `5000`
