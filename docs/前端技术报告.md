# 前端技术报告

---

## 一、项目概述

本项目"智创电商营销系统"采用双前端架构：
- **Next.js应用**（端口3000）：现代化登录认证界面
- **Flask模板**（端口5000/5001）：传统业务功能界面

## 五、前后端通信接口

### 5.1 服务端口分布

| 服务 | 端口 | 职责 |
|------|------|------|
| Next.js前端 | 3000 | 登录页面UI |
| Login Server | 3002 | 登录API服务 |
| AIGC Server | 5000 | 主业务服务 |
| App Server | 5001 | 数据看板服务 |

### 5.2 登录页面API（Next.js → Login Server:3002）

| API | 方法 | 请求参数 | 响应格式 |
|-----|------|----------|----------|
| `/api/login` | POST | `{email, password}` | `{status: 'success', message: '...'}` |

**登录流程**：
1. 用户在Next.js页面输入账号密码
2. POST到`localhost:3002/api/login`
3. 成功后重定向到`localhost:5000/home`

### 5.3 数据看板API（dashboard.html → App Server:5001）

| API | 方法 | 请求参数 | 响应格式 |
|-----|------|----------|----------|
| `/api/dashboard/data` | GET | `timeRange, type, platforms, last_update` | 聚合数据对象 |
| `/api/auth/{platform}` | GET | - | `{auth_url: '...'}` |
| `/api/check_auth/{platform}` | GET | - | `{authorized: boolean}` |
| `/api/dashboard/generated_content` | GET | - | `{status, tasks: [...]}` |

**看板数据响应结构**：
```json
{
  "update_time": "2025-03-22 10:30:00",
  "total_revenue": 150000,
  "total_profit": 45000,
  "revenue_growth": "12.5%",
  "profit_growth": "8.3%",
  "revenue_distribution": [{"name": "京东", "value": 50000}],
  "profit_distribution": [{"name": "抖音", "value": 15000}],
  "revenue_profit": [{"revenue": 1000, "profit": 300}],
  "product_sales": [{"name": "商品A", "platform": "京东", "sales": 100}]
}
```

### 5.4 文案智造器API（test1.html → AIGC Server:5000）

| API | 方法 | 请求参数 | 响应格式 |
|-----|------|----------|----------|
| `/generate_xiaohongshu_stream_post` | POST | `{query}` | SSE流式响应 |
| `/generate_xiaohongshu` | POST | `{query}` | `{content, hashtags}` |
| `/submit-form-data` | POST | `{product_name, ad_best, copy_type}` | `{status, message}` |

**SSE流式响应格式**：
```
data: {"chunk": "文案片段..."}
data: {"done": true, "hashtags": "#标签1 #标签2"}
```

### 5.5 营销图创作API（test2.html → AIGC Server:5000）

| API | 方法 | 请求参数 | 响应格式 |
|-----|------|----------|----------|
| `/api/oss/categories` | GET | - | `{code, data: [...]}` |
| `/api/oss/products_by_category` | GET | `category` | `{status, data: [...]}` |
| `/api/generate_img2img` | POST | `{image_url}` | `{code, data: {task_id}}` |
| `/api/check_task_status/{task_id}` | GET | - | `{status, task: {...}}` |
| `/api/tasks/image` | GET | - | `{status, tasks: [...]}` |
| `/api/save_generated_content` | POST | `{product_id, type, file_url}` | `{status, oss_url}` |

**图片生成流程**：
1. GET `/api/oss/categories` 加载分类
2. GET `/api/oss/products_by_category` 加载商品
3. POST `/api/generate_img2img` 提交生成任务
4. GET `/api/check_task_status/{task_id}` 轮询状态
5. POST `/api/save_generated_content` 上传到OSS（可选）

## 八、总结

本项目前端采用双架构设计：
- **Next.js**提供现代化登录体验
- **Flask模板**快速实现复杂业务功能

前端与后端通过RESTful API通信，支持流式响应（SSE）用于AI内容生成。数据看板使用ECharts实现可视化，商品管理集成阿里云OSS存储。
